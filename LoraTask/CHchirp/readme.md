# 采样文件内容
```
位置:        采样文件放置于实验室NAS（\\192.168.88.1）\public\CHchirp中   
文件夹命名:   信道数目+channel+preamble信道位置（信号第一段的位置）  例子：4channel3 --> 总共4信道，位于该文件夹下的信号preamble都在信道3中
文件命名：    信道数目+channel+信道选择方案+_文件序号.sigmf-data    例子：4channel3210_15.sigmf-data -->  总共4信道，该文件信号的信道分段方式为[3,2,1,0],同信道选择方案文件中第15号文件
Ps：由于需要裁剪挑选正确的信号，文件序号并不是连续的，序号范围为0~19，但是可能会缺失其中某些序号的文件（平均每组文件有15个）
```
